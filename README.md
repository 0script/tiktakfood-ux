* Having nodejs
* Having npm 
* Having vue-cli
* Running the backend 

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/0script/tiktakfood-ux.git
   ```
2. Install the package
   ```shell
   $cd tiktakfood-ux/
   $npm install
   ```
3. Run the application
   ```shell
    $npm run serve
   ```
